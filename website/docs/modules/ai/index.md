# AI Modules

Forage integrates with [LangChain4j](https://docs.langchain4j.dev/) to provide auto-configured AI capabilities for Apache Camel routes.

## Components

- [Agents](agents.md) — Composable AI agents with tool use and guardrails
- [Chat Models](chat-models.md) — LLM providers: Ollama, OpenAI, Gemini, Anthropic, and more
- [Embeddings](embeddings.md) — Text embedding models
- [Vector Databases](vector-databases.md) — Embedding stores for RAG
- [Chat Memory](chat-memory.md) — Conversation storage: in-memory, Redis, Infinispan

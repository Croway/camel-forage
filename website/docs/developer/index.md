## Contributing to Documentation

To preview the documentation site locally:

```bash
cd website
pip install -r requirements.txt
mkdocs serve
```

Then open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser. Changes are reloaded automatically.
